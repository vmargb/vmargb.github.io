Post Title Goes Here — Make It Interesting

14 Dec 2025 · 6 min read · Umar Sharif

Start with something that earns the click. A question, a claim, a problem you ran into. Don't ease in — just begin.

A section heading

Section headings use Inconsolata in all-caps so they feel like terminal output labels. Use them to break up long posts, not as decoration for every paragraph.

Inline code looks like this. It keeps the monospace feel even inside the serif body text. 

# A code block
def hello():
    print("this is a code block")
    # it has a purple left border
    # and monospace text throughout

Use blockquotes sparingly — for actual quotes or key takeaways you want to stand out.

"The tools you use shape the way you think." — someone, probably.

This is a callout box. Good for notes, warnings, or things you want the reader to notice without breaking the flow.

Another section

Keep writing. The line-height and max-width are tuned to sit around 65–70 characters per line, which is the comfortable reading range. Don't change the max-width.

An hr is a good way to signal a topic shift without a heading. Use it when the tone or direction changes.

A smaller subheading

h3 uses Lora serif so it blends with body text but still creates visual hierarchy. Use it for subsections within a section — don't go deeper than h3.

That's the template. Delete everything inside .post-body and start writing.
